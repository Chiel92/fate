selectors Module¶

A selector is a function that is decorated by selector, either directly or indirectly. It is an action that is used to modify that selection of the session. We distinguish between functions that work selection-wise (global selectors) and function that work interval-wise (local selectors). Furthermore we have selectors that are based on regular expressions.
